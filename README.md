# DuelGame

A simple game backend logic built with JavaScript and TypeScript.

## About

DuelGame is a project featuring game mechanics for a dueling system. This repository contains the core logic for handling game interactions, player matches, and battle mechanics.

## Project Structure

The project is organized as follows:
- **JavaScript files** - Core game logic and mechanics implementation
- **TypeScript files** - Type-safe utilities and reusable components

## Getting Started

To get started with this project:

1. Clone the repository
2. run either

```JavaScript
node Duel.js
```
or with change of ability

```JavaScript
node Duel.js --per-round
```
## Technologies

- **JavaScript** - Core game logic
- **TypeScript** - Type-safe components and utilities
